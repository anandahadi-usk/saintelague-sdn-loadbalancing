DESCRIPTION:
This dataset contains the complete experimental results from a
controlled empirical study comparing four weighted load balancing
algorithms (WRR, IWRR, WLC, Sainte-Laguë) in Software-Defined
Networking (SDN) environments under dynamic weight-change conditions.

The dataset includes:
- 219 controlled experimental runs across 3 scenarios (S1/S2/S3)
- Per-flow MAPE trajectories for all 4 algorithms
- Convergence analysis: tau values and sustained convergence metrics
- QoS measurements: throughput, latency, jitter, packet loss
- Statistical test results: Mann-Whitney U, Cliff's delta
- Publication figures (300 DPI PNG, IEEE standard)
- Algorithm source code (Python, Ryu OpenFlow 1.3)

Key finding: Sainte-Laguë achieves sustained convergence (tau=88.8 flows,
107 consecutive flows below 5% MAPE), with complete stochastic dominance
(Cliff's delta=1.000) over all competitors. WLC exhibits Greedy Overshoot
failure causing -10.7% throughput degradation.

Platform: Mininet 2.3 + Ryu 4.34 + OpenFlow 1.3 + Python 3.8
