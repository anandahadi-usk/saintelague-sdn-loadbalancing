ABSTRAK / KETERANGAN
----------------------------------------------------------------------
This foundational technical report introduces the Jain Fairness Index (JFI), a quantitative measure to evaluate fairness in resource allocation. JFI = (sum xi)^2 / (n * sum xi^2) ranges from 1/n (maximally unfair) to 1 (perfectly fair). The measure is continuous, bounded, independent of scale, and applicable to any resource sharing problem including network bandwidth allocation and server load distribution.
