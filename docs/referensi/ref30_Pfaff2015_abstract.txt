ABSTRAK / KETERANGAN
----------------------------------------------------------------------
This paper describes the architecture and implementation of Open vSwitch (OVS), the production-grade virtual switch used as the data plane in the P5 testbed. OVS supports OpenFlow 1.3, enabling the Ryu controller to install and modify flow table entries that implement the DNAT/SNAT load balancing rules. The paper details OVS's flow caching, port mirroring, and QoS mechanisms — all of which are used in the P5 experimental infrastructure. OVS received the USENIX NSDI Best Paper Award in 2015.
