Judul  : Efficient fair queuing using deficit round-robin
Penulis: Shreedhar, M.; Varghese, G.
Jurnal : IEEE/ACM Transactions on Networking
Tahun  : 1996
DOI    : 10.1109/90.502236

ABSTRAK / KETERANGAN
----------------------------------------------------------------------
This seminal paper introduces Deficit Round Robin (DRR), which improves upon WRR by handling variable-length packets without per-packet normalisation overhead. DRR maintains a deficit counter per queue and grants service when accumulated deficit exceeds packet size, achieving O(1) complexity per packet. The interleaved variant (IWRR) used in P5 derives from DRR's sequence-building approach. Critically, this paper assumes static weights throughout its analysis — the dynamic weight-change scenario that motivates the P5 research is not considered.
