Judul  : A network in a laptop
Penulis: Lantz, Bob; Heller, Brandon; McKeown, Nick
Jurnal : Proceedings of the 9th ACM SIGCOMM Workshop on Hot Topics in Networks
Tahun  : 2010
DOI    : 10.1145/1868447.1868466

ABSTRAK / KETERANGAN
----------------------------------------------------------------------
This paper introduces Mininet, a network emulator that enables rapid prototyping of SDN architectures on a single machine. Mininet creates virtual hosts, switches, and links using Linux network namespaces and virtual Ethernet pairs, with tc-netem for bandwidth and delay emulation. The P5 testbed uses Mininet 2.3 with Open vSwitch to emulate 10 client hosts, one OpenFlow 1.3 switch, and three servers with bandwidth-limited links, following the architecture described in this paper.
